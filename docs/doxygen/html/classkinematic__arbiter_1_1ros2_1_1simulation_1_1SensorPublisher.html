Detailed Description

template<typename MsgType>
class kinematic_arbiter::ros2::simulation::SensorPublisher< MsgType >

Templated sensor publisher that handles different sensor types.

Publishes four variants of each sensor:

  • Raw (noisy measurements)
  • Truth (ground truth)
  • Upper bound (truth + 3*sigma)
  • Lower bound (truth - 3*sigma)
