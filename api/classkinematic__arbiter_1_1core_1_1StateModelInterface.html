inline void UpdateProcessNoise(const StateVector &a_priori_state, const StateVector &a_posteriori_state, double process_to_measurement_ratio, double dt)

Update process noise based on state correction.

Implements dynamic process noise estimation using the equation: Q̂_k = Q̂_{k-1} + (ζ·dt·(x̂_k^- - x̂_k^+)(x̂_k^- - x̂_k^+)^T - Q̂_{k-1})/n

Where:

  • Q̂_k is the new process noise estimate

  • ζ is the process to measurement noise ratio

  • dt is the time step

  • x̂_k^- is the a priori state estimate (before update)

  • x̂_k^+ is the a posteriori state estimate (after update)

  • n is the sample window size

Parameters:

  • a_priori_state – The a priori state estimate x̂_k^-

  • a_posteriori_state – The a posteriori state estimate x̂_k^+

  • process_to_measurement_ratio – The process to measurement noise ratio (ζ)

  • dt – Time step in seconds
