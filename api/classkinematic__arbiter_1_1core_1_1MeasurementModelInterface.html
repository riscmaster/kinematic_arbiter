class MeasurementModelInterface

Interface for measurement models with assumption validation.

Provides methods for predicting measurements, computing Jacobians, and validating filter assumptions using chi-squared testing.

Template Parameters:

Type – Sensor type that defines the measurement vector dimensions

Subclassed by kinematic_arbiter::sensors::BodyVelocitySensorModel, kinematic_arbiter::sensors::ImuSensorModel, kinematic_arbiter::sensors::PoseSensorModel, kinematic_arbiter::sensors::PositionSensorModel
