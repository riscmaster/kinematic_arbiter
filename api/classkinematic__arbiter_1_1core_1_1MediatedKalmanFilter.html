template<int StateSize, typename ProcessModel>
class MediatedKalmanFilter

MediatedKalmanFilter implements a Kalman filter with chi-squared validation.

Leverages StateModelInterface for prediction and ProcessNoise management and MeasurementModelInterface for measurement processing, validation and mediation.

Template Parameters:

  • StateSize – Dimension of state vector

  • ProcessModel – Type implementing StateModelInterface
