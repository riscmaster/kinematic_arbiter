class PoseSensorModel : public kinematic_arbiter::core::MeasurementModelInterface

7DOF pose measurement model (position + orientation)

Models a sensor that measures position and orientation in 3D space. Measurement vector is [x, y, z, qw, qx, qy, qz]’ where [qw, qx, qy, qz] represents orientation as a quaternion.
