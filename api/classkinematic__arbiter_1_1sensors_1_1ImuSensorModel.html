class ImuSensorModel : public kinematic_arbiter::core::MeasurementModelInterface

IMU measurement model (gyroscope + accelerometer)

Models an IMU sensor that measures angular velocity and linear acceleration. Measurement vector is [gx, gy, gz, ax, ay, az]’ where [gx, gy, gz] represents angular velocity and [ax, ay, az] represents linear acceleration.
